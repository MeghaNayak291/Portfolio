# Personal Portfolio Website

This is my personal portfolio website showcasing my work, skills, and projects.

The website is designed with a clean and modern layout, focusing on simplicity, usability, and responsive design.

## Features

- Clean and modern interface
- Fully responsive design
- About section
- Projects section
- Contact section
- Smooth navigation

## Tech Stack

- HTML
- CSS
- JavaScript

## Live Preview

You can view the live website here:
https://portfolio-livid-theta-87.vercel.app/

## How to Run Locally

1. Clone the repository:
   git clone https://github.com/MeghaNayak291/Portfolio.git

2. Open the project folder.

3. Open index.html in your browser.
